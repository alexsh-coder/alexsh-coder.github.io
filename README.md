# MD → книжечка для печати

Инструмент превращает список вопросов к экзамену в готовую к печати мини-книжку с карточками. Печатается на обычном A4, складывается и склеивается в книжечку.

---

## Как пользоваться

### Шаг 1 — Получить ответы от нейросети

Скопируй все вопросы к экзамену и отправь в ChatGPT, Claude или любую другую нейросеть **вот с таким промптом:**

---

```
Я готовлюсь к экзамену. Ниже список вопросов.
Создай Markdown-файл для шпаргалки-книжечки.

Формат файла строго такой:
- Первая строка файла: # Шпаргалка
- Каждый вопрос — отдельный блок, разделённый строкой ---
- Заголовок блока: ## N) Название вопроса  (N — номер по порядку)
- Дальше — короткий, но полный ответ: определения, ключевые пункты, примеры кода если нужно
- Блоки кода оборачивай в ```язык ... ```
- Жирным (**текст**) выделяй самое важное
- Ответ должен быть компактным — он должен уместиться в маленькую карточку

Вот вопросы:
[ВСТАВЬ СЮДА СВОИ ВОПРОСЫ]
```

---

Нейросеть вернёт готовый `.md` файл. Скопируй его содержимое и сохрани как файл с расширением `.md` (например, `shpargalka.md`).

> **Совет:** если ответы получились слишком длинными — попроси нейросеть сократить: *«Сделай ответы короче, каждый не больше 5-7 строк»*

---

### Шаг 2 — Открыть инструмент

Зайди на **[alexsh-coder.github.io](https://alexsh-coder.github.io)**

---

### Шаг 3 — Загрузить файл

Перетащи `.md` файл в зону **«Перетащи .md сюда»** или нажми на неё и выбери файл.

Справа сразу появится предпросмотр — ровно так, как это будет выглядеть на бумаге. В левой панели отобразится статистика: сколько карточек и сколько листов A4 нужно напечатать.

---

### Шаг 4 — Напечатать

Нажми кнопку **«Печать прямо сейчас»**.

В диалоге печати выставь:
- Размер бумаги: **A4**
- Поля: **минимальные** (или «нет»)
- Масштаб: **100%**
- Колонтитулы: **выключить**
- Фоновая графика: **включить** (иначе серый фон кода не пропечатается)

---

### Шаг 5 — Сложить книжечку

1. Разрежь лист по пунктирным линиям на 16 равных карточек
2. Сложи в порядке номеров страниц
3. Склей соседние листы по меткам **«место склейки»**

---

## Дополнительно

**Скачать файл для печати** — кнопка «↓ Скачать HTML для печати» сохранит готовый файл, который можно открыть позже или распечатать на другом устройстве.

**Размер текста** — если текст не помещается в карточки, подвинь ползунок влево, чтобы уменьшить шрифт.

**Линии сгиба** — можно отключить пунктир, если хочешь чистый вид без разметки.

**Правка на лету** — нажми «показать / править исходник» и редактируй Markdown прямо в браузере, предпросмотр обновляется моментально.
